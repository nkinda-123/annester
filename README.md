# Annester Kibonde Portfolio

This project contains a personal portfolio website for a data science student. It includes:

- `frontend/` — a modern responsive portfolio site built with HTML, CSS, and JavaScript
- `backend/` — a simple Express API with portfolio profile and projects data

## Deployment

### Frontend
- Host `frontend/` on Vercel as a static site
- Set the frontend `apiBase` in `frontend/js/app.js` to your deployed backend URL if needed

### Backend
- Host `backend/` on Render as a Node.js service
- Use `npm install` and `npm start`
